# 📊 Technology Trends Analysis using Job Market & Developer Survey Data

## 🔎 Project Overview

This capstone project analyzes global technology trends by combining job market data, developer survey responses, and salary datasets to identify:

- Current in-demand technologies
- Future growth trends
- Salary comparisons across programming languages
- Database adoption patterns
- Global participation insights

The goal is to provide data-driven insights that help students, job seekers, developers, and organizations make informed technology and career decisions.

---

## 🎯 Objectives

- Analyze job postings to determine market demand
- Evaluate developer survey data to identify usage trends
- Compare salary distributions across programming languages
- Study database popularity and future growth
- Build an interactive dashboard for visual insights

---

## 📂 Data Sources

1. **Job Postings Dataset** – Technology demand analysis  
2. **Web-Scraped Salary Dataset** – Salary comparison by language  
3. **Stack Overflow Developer Survey (Processed)** – Usage & future trends

---

## 📊 Key Findings

### 💻 Programming Languages

**Current Trends**
- JavaScript leads in job postings and developer usage
- HTML/CSS and SQL remain essential
- Python is among the top 4 most used languages

**Future Trends**
- Python is projected to become the most desired language
- SQL remains highly relevant
- Rust and Go show strong growth potential

---

### 🗄 Database Trends

**Current**
- PostgreSQL leads overall preference
- MySQL and SQLite remain widely used
- SQL Server maintains strong presence

**Future**
- PostgreSQL remains dominant
- Redis and SQLite show growth
- Supabase appears as an emerging database

---

### 💰 Salary Insights

- Swift shows the highest average salary
- Python and C++ offer strong compensation
- Java and Go remain competitive
- High salary does not always correlate with highest demand

---

### 🌍 Global Participation

- United States has the highest number of respondents
- Germany and India follow
- Europe and North America dominate survey participation
- Data reflects broad global technology adoption patterns

---

## 📈 Dashboard Overview

An interactive dashboard was built using IBM Cognos Analytics with three main sections:

### 1️⃣ Current Technology Usage
- Most used programming languages
- Most preferred IDEs
- Leading database technologies

### 2️⃣ Future Technology Trends
- Most desired programming languages
- Future database preferences
- Emerging technologies

### 3️⃣ Demographics
- Country-wise participation
- Regional distribution of developers

The dashboard provides clear, visual insights that support data-driven decision-making.

---

## 📌 Overall Insights & Implications

### Findings
- Python is the fastest-growing language
- PostgreSQL dominates database preference
- JavaScript remains industry standard
- Backend and data-oriented technologies are steadily increasing

### Implications
- Professionals should prioritize Python + SQL skills
- Organizations should adopt scalable open-source databases
- Developers should monitor emerging technologies like Rust and Go
- Continuous learning is essential in the tech industry
